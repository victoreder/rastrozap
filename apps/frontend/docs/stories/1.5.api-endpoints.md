## 📋 Description

Create API endpoints for authentication workflows. These endpoints will handle user registration, login, and logout with Supabase Auth integration.

## 🎯 Endpoints to Build

### 1. POST /api/auth/signup
- **Purpose:** Register new user account
- **Request Body:** { email, password, name }
- **Response:** { user, session }
- **Status Codes:** 201 (success), 400 (validation), 500 (error)

### 2. POST /api/auth/login
- **Purpose:** Authenticate user and create session
- **Request Body:** { email, password }
- **Response:** { user, session }
- **Status Codes:** 200 (success), 400 (validation), 401 (auth failed), 500 (error)

### 3. POST /api/auth/logout
- **Purpose:** Destroy session and logout user
- **Response:** { success: boolean }
- **Status Codes:** 200 (success), 401 (not auth), 500 (error)

### 4. GET /api/auth/me
- **Purpose:** Get current user info (protected)
- **Response:** { user, session }
- **Status Codes:** 200 (success), 401 (not auth), 500 (error)
