# Story 1.7: Landing Page

**Status:** Draft
**Epic:** 1 (Foundation & Auth)
**Priority:** High
**Points:** 5
**Created:** 2026-02-21

---

## 📋 Description

Create a marketing landing page that introduces RastroZap to potential customers. The page should clearly communicate the value proposition, show key benefits, and provide a call-to-action to start a free trial or login.

---

## ✅ Acceptance Criteria

- [ ] Landing page route at `/` (public, no auth required)
- [ ] Hero section with headline, subheadline, and CTA button
- [ ] Problem/Solution section explaining the "black hole" of lead data
- [ ] Features section highlighting core benefits (tracking, automation, insights)
- [ ] Pricing section showing the Starter plan (R$97/month, 500 leads/day)
- [ ] FAQ section addressing common questions
- [ ] Footer with links and branding
- [ ] Responsive design (mobile-first, 375px to 1440px)
- [ ] All interactive elements tested (buttons, links, forms)
- [ ] Page loads in < 2s (Lighthouse score > 80)

---

## 🎯 Implementation Details

### Hero Section
- Heading: "Rastreie Leads do WhatsApp com Precisão"
- Subheading: "Saiba exatamente qual anúncio trouxe cada lead"
- CTA Button: "Começar Grátis" → `/registro`
- Background: Subtle gradient or pattern (color #014FBB based)

### Problem/Solution Section
- Problem: "Você investe em anúncios mas não sabe quantos leads reais chegam"
- Solution: "RastroZap rastreia cada clique e vincula ao seu WhatsApp"
- Visual: 3-column layout with icons (click → track → analyze)

### Features Section
- Feature 1: "Links Rastreáveis" - Short codes with UTM tracking
- Feature 2: "Jornada Automática" - Track leads through conversion funnel
- Feature 3: "Relatórios Acionáveis" - Real-time dashboards and metrics
- Feature 4: "Integração WhatsApp" - Direct UazAPI connection
- Each feature has icon (SVG), title, and brief description

### Pricing Section
- Plan: "Starter"
- Price: "R$ 97/mês"
- Capacity: "500 leads/dia"
- Includes: All core features, email support
- CTA: "Comece Agora" button

### FAQ Section
- 5-8 common questions
- Accordion component for collapse/expand
- Topics: pricing, features, integration, support, etc.
