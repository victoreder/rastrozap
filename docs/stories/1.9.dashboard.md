# Story 1.9: Dashboard

**Status:** Draft
**Epic:** 1 (Foundation & Auth)
**Priority:** High
**Points:** 8
**Created:** 2026-02-21

## 📋 Description

Build the main dashboard page that serves as the central hub for RastroZap users. The dashboard displays key metrics, recent activity, and navigation to core features (links, leads, settings).

---

## ✅ Acceptance Criteria

- [ ] Dashboard page at `/app/dashboard` (protected route)
- [ ] Page layout: Header + Sidebar + Content area
- [ ] Header with logo, user menu, logout button
- [ ] Sidebar with navigation links (Dashboard, Links, Leads, Jornada, Configurações)
- [ ] Metrics cards showing (total links, total leads, leads today, conversion rate)
- [ ] Recent activity widget (last 5 leads/clicks)
- [ ] Quick action buttons (Create Link, View Reports)
- [ ] Welcome message with user's name
- [ ] Empty state when no data available
- [ ] Responsive design (mobile sidebar collapses)
- [ ] Real-time updates of metrics (optional)
- [ ] Dashboard tests covering key components
- [ ] Integration with Supabase to fetch account data

---

## 🎯 Implementation Details

### Page Layout
- Header: Fixed top, contains logo, breadcrumbs, user menu
- Sidebar: Fixed left (collapsible on mobile), navigation items
- Content: Main area with dashboard widgets

### Header Component
- Logo/Home link on left
- Center: Breadcrumb or page title
- Right: User menu dropdown (Profile, Settings, Logout)
- Icons: Bell (notifications), Settings, Profile avatar

### Sidebar Component
- Logo at top
- Navigation items:
  - Dashboard (icon + "Dashboard")
  - Links (icon + "Meus Links")
  - Leads (icon + "Leads")
  - Journey (icon + "Jornada")
  - Settings (icon + "Configurações")
- Collapse button (mobile)
- Footer: User info or upgrade button

### Dashboard Content
- Welcome section: "Bem-vindo, [User Name]!"
- Metrics cards (4 total):
  - Card 1: "Links Criados" - count
  - Card 2: "Leads Rastreados" - count
  - Card 3: "Leads Hoje" - count
  - Card 4: "Taxa de Conversão" - percentage
- Recent Activity widget:
  - Table/List of last 5 items
  - Columns: Data, Link, Lead, Status
- Quick Actions:
  - "Criar Novo Link" button
  - "Ver Relatórios" button

### Responsive Behavior
- Desktop: Sidebar visible, 3-column metrics
- Tablet: Sidebar collapses to hamburger
- Mobile: Sidebar hidden, hamburger menu toggle
