- File: `apps/frontend/src/app/(auth)/login/page.tsx`
- Form fields: Email, Password
- Button: "Entrar" (calls `/api/auth/login`)
- Links: "Criar conta" в†’ `/registro`, "Esqueci senha" в†’ `/esqueci-senha`
- Error display: Toast or inline error message
- Success: Redirect to `/app/dashboard`
- Remember me: Optional (store in localStorage)

### Register Page (`/registro`)
- File: `apps/frontend/src/app/(auth)/registro/page.tsx`
- Form fields: Email, Password, Password Confirm
- Button: "Criar Conta" (calls `/api/auth/signup`)
- Links: "JГЎ tem conta?" в†’ `/login`
- Terms: Checkbox for terms acceptance
- Error display: Field-level + form-level errors
- Success: Redirect to `/app/dashboard` with user session

### Form Validation
- Email: Valid email format (RFC 5322)
- Password: Minimum 6 characters
- Password Confirm: Must match Password field
- Terms: Must be checked on register

### Error Handling
- Network error: Show retry message
- Invalid credentials: "Email ou senha incorretos"
- Account exists: "Email jГЎ cadastrado"
- Weak password: "Senha muito fraca (mГ­nimo 6 caracteres)"

### Layout
- Centered card design (max-width 400px)
- Logo/branding at top
- Form with all fields visible at once
- Submit button full width
- Links to alternate pages
