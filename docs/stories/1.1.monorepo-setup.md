## 📖 Description

This story establishes the foundational monorepo setup that all other stories depend on. The monorepo enables:
- Shared types between frontend and backend (`packages/tipos-compartilhados`)
- Coordinated development and builds via Turborepo
- Single source of truth for dependencies
- Efficient parallel testing and linting

**Scope:**
- Create directory structure
- Configure npm workspaces
- Set up Turborepo
- Initialize git and .gitignore
- Create .env.example
- Document setup in README

### npm Workspaces Configuration

**Root package.json:**
```json
{
  "name": "rastrozap-monorepo",
  "version": "0.1.0",
  "description": "WhatsApp lead tracking platform for traffic managers",
  "workspaces": [
    "apps/frontend",
    "apps/backend",
    "packages/tipos-compartilhados"
  ],
  "scripts": {
    "dev": "turbo run dev",
    "build": "turbo run build",
    "lint": "turbo run lint",
    "typecheck": "turbo run typecheck",
    "format": "prettier --write .",
    "clean": "turbo run clean && rm -rf node_modules"
  },
  "devDependencies": {
    "turbo": "^2.0.0",
    "prettier": "^3.0.0",
    "eslint": "^8.0.0"
  }
}
```

### Turborepo Configuration

**turbo.json:**
```json
{
  "extends": ["//"],
  "globalDependencies": [".eslintrc.json", "prettier.config.js"],
  "pipeline": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**", ".next/**", "build/**"],
      "cache": true
    },
    "dev": {
      "cache": false,
      "persistent": true
    },
    "lint": {
      "outputs": [],
      "cache": true
    },
    "typecheck": {
      "outputs": [],
      "cache": true
    },
    "test": {
      "outputs": ["coverage/**"],
      "cache": true
    }
  }
}
```
